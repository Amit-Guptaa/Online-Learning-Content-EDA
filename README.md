# Online Learning Platform Content Upload Analysis

## Project Type
Minor-1 | Exploratory Data Analysis (EDA)

## Objective
Analyze content upload data from an online learning platform to identify trends over time using summary statistics and bar plot visualizations.

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib

## Features
- Data cleaning and preprocessing
- Monthly trend detection
- Category-wise analysis
- Bar plot visualizations

## Dataset
The dataset contains:
- content_id
- upload_date
- category
- instructor
- duration_minutes
- views

## Conclusion
The analysis shows a positive growth trend in content uploads, with Programming being the most dominant category.
